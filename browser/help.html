Help

The project aims to build a low-cost, open-source wearable device that can be used in swimming practice. It works with prototype wristbands that collect inertial signals, that, once teamed up with a testes data analysis system, can determine the swimming style and identify errors in the stroke execution by comparing the data with reference values as well as make an overall assessment of the athlete's performance. With an accuracy rate of around 95% in detecting different swimming styles, this project offers a cheaper and more accessible option compared to the swimming devices currently available in the market, which tend to be more expensive due to the need for waterproofing and performing extensive analyses.

The project consists of 3 parts: Web, Mobile and Hardware/Sensor. Follow the README especifications on the repositories listed below:

  • Web
  • Mobile
  • Sensor
